# Alerting
Чтобы вызвать alert выполните команды
```sudo apt install hey```
```hey -z 1m -c 10 http://51.250.40.137:80/error```
Эта команда создаст нагрузку на приложение в течение 1 минуты с 10 одновременными соединениями.
